                          # 前言

                          您已经听说过很多有关使用异步或并发编程的信息。现在是时候让您了解更多关于协程以及它们如何在后台工作的知识了。

                          在本章中，您将了解以下内容：

                          • 了解例程[1]以及程序如何控制其执行流程。
                          • 在代码中了解挂起函数与挂起点。
                          • 启动您的第一个 Kotlin 协程。
                          • 通过创建一些典型的任务来练习您所学的内容，例如切换到 UI 线程。
