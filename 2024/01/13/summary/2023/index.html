2023年终总结

回首2023年，真切的感觉时间过的真快，古人云“如白驹过隙”，诚不欺我。

文章内容输出

文章内容主要有以下几个方面：

  • Kotlin
  • Kotlin协程
  • OpenHarmony
  • Jetpack Compose
  • Servlet
  • Node CLI
  • MavenCentral

视频内容输出

仅有一个视频内容输出，不提也罢😅。

开源项目

2023年在开源项目上投入的精力不多，大多数是在10月份之后，主要是维护以下两个项目。

  • Mask

    • 适配 AGP 8.0
    • 适配 Kotlin K2 Compiler

  • Sword

    • 适配 Kotlin K2 Compiler

总结2023

立下的 Flags 完成度总是差强人意。

2023又是懈怠的一年，Spring 没有继续学习，SpringMVC 和 MyBatis 就更不用提了，唯一欣慰的是在 OpenHarmony 和 Compose 上有点产出，挖的开源项目的坑也有在填。

总结起来，归根结底还是自驱力不行。

展望2024

计划总是赶不上变化，但是没有计划就是漫无目的的乱飞，所以还是列个小目标，万一实现了呢。

Java

Java 还是延续2023年的目标吧，希望可以坚持下去。

  1. 更新完 Spring 学习日记，
  2. 更新 SpringMVC 学习日记，
  3. 更新 MyBatis 学习日记

OpenHarmony

手头没有了 OpenHarmony 物理设备，OpenHarmony 暂停一下，接下来可能会关注华为的 HarmonyOS Next。

明年公司可能会转到鸿蒙上面，所以明年可能会更新：

  1. TypeScript 学习日记
  2. OpenHarmony 应用开发

Compose

一直对 Dart 语言不感冒，所以对于 Flutter 的兴趣也不大，导致 Flutter 一直没有坚持学习下去，接下来打算放弃对 Flutter 的学习，转而坚持学习 Compose，随着 Compose Multiplatform 的稳定，Compose 的跨平台能力也有很大提升。

最后

2024 继续加油。
