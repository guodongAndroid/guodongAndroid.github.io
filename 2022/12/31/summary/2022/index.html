2022年终总结

2022年终总结

疫情

今年年初疫情“爆发”，当时正值在总公司出差，公司为了同事们的安全着想，号召同事们在公司内办公，同时公司买进了一些床垫、被褥枕头等，还发了个人洗漱用品，就此开始了在公司与同事们吃住办公三个多周。现在回想起来这段与大家共度时艰的经历也是一段美好的回忆。

文字内容输出

做 Android 开发多年，2016年搭建了个人博客，不过一直是荒废状态，正好年初疫情时期在公司内有空闲时间，就把个人博客收拾了一下，重新开始输出，同时也在 掘金 和个人公众号「guodongAndroid」上发布。

文章内容主要与以下几个方面相关：

  • linphone-sdk-android
  • kotlin
  • android

视频内容输出

视频内容输出纯属巧合吧。下半年在学习 Java 服务端开发，为加强学习记忆，开始时也是以文章的形式输出，后面发现以文章的形式输出较为繁琐并且有些地方以文字的形式描述起来不够通透，存在表述不清的情况，所以考虑以视频的形式输出。

目前视频主要是 Spring 学习日记：

  • 《Spring学习日记》更新中，共 6 集

展望2023

明年可能会主要以视频的形式输出，文字内容输出应该也会有。

Java

  1. 更新完 Spring 学习日记，
  2. 更新 SpringMVC 学习日记，
  3. 更新 MyBatis 学习日记

OpenHarmony

明年公司可能会转到鸿蒙上面，所以明年可能会更新：

  1. TypeScript 学习日记
  2. OpenHarmony 应用开发

Flutter & Compose

之前学过一阵子 Flutter 和 Compose，却是没能坚持下来，希望明年能坚持学习下来。

最后

欢迎大家在 B 站关注我：孫小逗

同时也欢迎大家关注微信公众号：「guodongAndroid」。

2023 继续加油。
